Editor de código

Editor de código/texto

Para qualquer modificação no código fonte do projeto será utilizado o Visual Studio Code por padrão. Por causa de sua simplicidade, extensibilidade e perfomance.

Lista de plugins

  • DotENV (mikestead)
  • EditorConfig for VS Code (EditorConfig)
  • ESLint (Dirk Baeumer)
  • GitLens - Git supercharged (Eric Amodio)
  • Todo Tree (Gruntfuggly)
  • vscode-icons (VSCode Icons Team)
  • ES7 React/Redux/GraphQL/React-Native snippets (dsznajder)

Submetendo novos plugins

Caso acredite que seja necessário adicionar mais algum plugin para a lista, envie um email para <\> com o nome do plugin, o criador e o motivo de ser necessário. Responderemos prontamente ao pedido.
